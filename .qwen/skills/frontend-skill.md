# Frontend Skill

## Objective
Build and maintain the user interface and user experience using modern web technologies, ensuring seamless integration with backend services.

## Domain
User Interface & User Experience

## Capabilities

### 1. Page and Layout Architecture
- Create nested layouts using Next.js 16+ App Router
- Build optimized page routes with proper loading states
- Implement responsive design principles
- Ensure accessibility compliance (WCAG standards)

### 2. Component Development
- Build reusable, accessible UI components using Tailwind CSS
- Implement proper component composition patterns
- Create consistent design system and styling
- Ensure components are properly typed with TypeScript

### 3. State Management and Logic
- Use React Server Components (RSC) by default for optimal performance
- Implement Client Components only where interactivity is required
- Manage application state effectively using React hooks
- Handle form state and user input validation

### 4. API Integration
- Connect UI components to Backend API via centralized fetch/client
- Handle authentication headers automatically in API calls
- Implement proper error handling for API responses
- Manage loading states and optimistic updates

### 5. Performance Optimization
- Implement code splitting and lazy loading
- Optimize images and assets
- Minimize bundle sizes
- Implement proper caching strategies

### 6. User Experience
- Create intuitive navigation and user flows
- Implement proper feedback for user actions
- Ensure fast load times and smooth interactions
- Handle edge cases and error states gracefully

## Integration Requirements
- Seamlessly connect with backend API endpoints
- Implement proper authentication flow with JWT tokens
- Follow consistent data models with backend and database
- Maintain responsive design across devices

## Quality Standards
- Follow modern React and Next.js best practices
- Ensure cross-browser compatibility
- Implement proper testing strategies
- Maintain clean, readable, and maintainable code
