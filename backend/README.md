# Secure Todo Application Backend

This is the backend for the Secure Todo Application, built with FastAPI and PostgreSQL.

## Features

- User authentication and authorization
- Todo management (create, read, update, delete)
- JWT-based security
- Async database operations with Neon PostgreSQL

## Tech Stack

- Python 3.11+
- FastAPI
- SQLModel
- Neon PostgreSQL
- JWT authentication
- bcrypt for password hashing
